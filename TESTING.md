## 💻 ローカルでプレイ

### 必要なもの
- Node.js 18以上
- Git

### 手順

1. **リポジトリをクローン**
   ```bash
   git clone [リポジトリURL]
   cd Phaser_Dev_Game_1
   ```

2. **依存関係をインストール**
   ```bash
   npm install
   ```

3. **開発サーバーを起動**
   ```bash
   npm run dev
   ```

4. **ブラウザで開く**
   - 自動的にブラウザが開きます
   - または http://localhost:3000 を開く

## 🎯 操作方法

### 基本操作
- **WASD** または **矢印キー**: 移動
- **マウス**: 照準
- **スペースキー** または **マウスクリック**: 射撃
- **1-5**: 武器切り替え
  - 1: Pistol（基本武器）
  - 2: Shotgun（散弾）
  - 3: Rifle（高速連射）
  - 4: Sniper（貫通・高火力）
  - 5: Laser（貫通）
- **R**: ゲームオーバー後の再スタート

### ゲームの目的
- 敵を倒してスコアを稼ぐ
- レベルアップしてステータス強化
- アイテムを集めてパワーアップ
- クレジットを貯める（永続保存）
